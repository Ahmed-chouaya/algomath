## Directory Purposes

**.opencode/get-shit-done/ (Primary Source)**
- Purpose: Main GSD framework implementation
- Contains: Workflows, templates, agents, CLI tools
- Key files: `workflows/*.md`, `templates/*.md`, `bin/gsd-tools.cjs`
- Subdirectories: `bin/`, `references/`, `templates/`, `workflows/`

**.claude/**
- Purpose: Claude Code integration
- Contains: Agent definitions, slash commands, hooks
- Key files: `agents/*.md`, `commands/gsd/*.md`
- Subdirectories: `agents/`, `commands/`, `get-shit-done/`, `hooks/`

**.github/**
- Purpose: GitHub integration and shared resources
- Contains: Agent definitions, workflow templates
- Key files: `agents/*.md`, `get-shit-done/templates/*.md`

**.codex/**
- Purpose: Codex CLI configuration
- Contains: Tool configuration
- Key files: `config.toml`

**.cursor/**
- Purpose: Cursor IDE integration
- Contains: Cursor-specific settings and agents

**.gemini/**
- Purpose: Gemini CLI integration
- Contains: Gemini-specific settings and agents

**.windsurf/**
- Purpose: Windsurf IDE integration
- Contains: Windsurf-specific settings and agents

**.planning/**
- Purpose: Project state and planning documents
- Contains: Config, roadmap, requirements, state
- Created by: GSD workflows
- Subdirectories: `codebase/`, `research/`, `phases/`

## Naming Conventions

**Files:**
- `kebab-case.md` - Markdown documents
- `gsd-*.md` - Agent/workflow files
- `UPPERCASE.md` - Important templates
- `*.cjs` - CommonJS scripts

**Directories:**
- `kebab-case` - All directories
- Plural for collections: `templates/`, `workflows/`, `agents/`
- Hidden for config: `.claude/`, `.planning/`

**Special Patterns:**
- `{command-name}.md` - Slash command definition
- `gsd-{role}.md` - Agent definition
- `*-template.md` - Template documents

## Where to Add New Code

**New Workflow:**
- Implementation: `.opencode/get-shit-done/workflows/{name}.md`
- Reference: Update command that calls it

**New Template:**
- Implementation: `.opencode/get-shit-done/templates/{name}.md`
- Usage: Reference from workflows

**New Agent:**
- Implementation: `.opencode/get-shit-done/agents/{name}.md`
- Also add to: `.claude/agents/{name}.md`, `.github/agents/{name}.md`

**New Command:**
- Implementation: `.opencode/commands/gsd/{name}.md`
- Also add to: `.claude/commands/gsd/{name}.md`

## Special Directories

**.planning/**
- Purpose: Project state and planning
- Source: Generated by GSD workflows
- Committed: Yes (configurable)

**Platform directories (.claude/, .cursor/, etc.)**
- Purpose: AI assistant specific configurations
- Source: Manual setup or GSD installation
- Committed: Yes
