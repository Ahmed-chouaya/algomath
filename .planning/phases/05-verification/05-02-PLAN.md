## Key Interfaces

### Algorithm and Step structure (from extraction/schema.py):
```python
@dataclass
class Algorithm:
    name: str
    description: str
    inputs: List[Dict[str, Any]]
    outputs: List[Dict[str, Any]]
    steps: List[Step]
    source_text: str

class StepType(Enum):
    ASSIGNMENT, LOOP_FOR, LOOP_WHILE, CONDITIONAL, RETURN, CALL, COMMENT

@dataclass
class Step:
    id: int
    type: StepType
    description: str
    inputs: List[str]
    outputs: List[str]
    condition: Optional[str]
    body: List[int]
    iter_var: Optional[str]
    expression: Optional[str]
```

### SandboxExecutor (from execution/sandbox.py):
```python
class SandboxExecutor:
    def execute(self, code: str, inputs: Optional[Dict] = None) -> ExecutionResult
```

- D-07: Show actual values from execution trace
- D-08: Match explanation style to algorithm complexity
- D-13: Static analysis for patterns (empty loops, division by zero, recursion)
- D-14: Execution-based edge case testing
- D-15: Report both detected and potential edge cases
- D-16: Edge cases include empty, single, max, negative, zero values
